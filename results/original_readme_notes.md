# Original README — Key Themes

**Title:** Social Dynamics — Particle Simulation

**Core framing:** "Preference-directed particle simulation exploring emergent complex dynamics from simple interaction rules."

**Key concepts from the original:**
- Particles have preference vectors that determine attraction/repulsion
- Interplay between movement, neighbor topology, and preference evolution
- Rich self-organizing behavior emerges from simple rules
- Signal/Response split — separate broadcast identity from reaction weights
- Social learning — conformity or differentiation
- Quiet-dim differentiation — differentiate along inactive dimensions
- Spatial memory field — persistent grid modulating preferences
- Force landscape visualization
- Shadow simulation for chaotic sensitivity
- Multiple physics engines (Numba, NumPy, PyTorch, Grid)
- Comprehensive math docs (simulator_overview.pdf, spatial_memory.pdf)

**Modules:** sim_2d_exp (main), sim_2d_cuda (CUDA), 3D_sim

**Spirit:** An exploration platform for emergent social dynamics from simple rules. Not task-oriented — it's about observing what happens when particles with preferences interact.
